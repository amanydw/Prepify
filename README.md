## 🧠 What is Prepify?

**Prepify** is a powerful all-in-one study tracking system designed especially for students preparing for competitive exams like **JEE / NEET / etc.**

It helps you:

* Track daily tasks 📋
* Log subject-wise practice 📊
* Analyze performance 📈
* Maintain consistency streaks 🔥
* Improve productivity using real data

---

## ⚡ Key Features

### 📋 Smart Task Management

* Add, complete & track tasks
* Date-based filtering (Today / Week / Month)
* Tag system for better organization
* Completion tracking with streak system

## 🧩 How It Works

* Data stored locally + synced to cloud
* Tasks & logs dynamically rendered
* Charts auto-update with user activity
* Streak system based on:

  * ≥10 Qs per subject
  * ≥5 tasks completed

---

## 🎯 Productivity System

Prepify uses a **points-based system**:

| Activity         | Points |
| ---------------- | ------ |
| Question Solved  | +100   |
| Task Completed   | +100   |
| Incorrect Answer | -25    |

👉 This creates a **real performance score**, not fake motivation.
