About This App

👋 Welcome to Prepify!

Your all-in-one academic productivity companion — built for serious JEE aspirants who want to track every question solved, every mock taken, every topic revised, and every task completed. Prepify keeps you accountable, consistent, and analytically sharp.

✅ 1. Smart Task Manager

A full-featured to-do system with date scheduling, #tag support, and date-based filters (Today, Tomorrow, This Week, This Month).

  • Separate To Do and Completed / Undone sections
  • +100 points when a task is marked done; −50 points when marked undone
  • No point change on deletion
  • Edit, duplicate, and delete tasks via options menu

📝 2. Daily Practice Logging

Log questions solved per day for Physics, Chemistry, and Maths. Entries accumulate so multiple logs in a day stack correctly.

  • Edit any past log entry with date correction
  • Full practice history with subject-wise breakdown
  • +5 points per question solved

🎯 3. Mock Test Tracker

Record full mock test data including name, date, time taken, max marks, correct, incorrect, and skipped questions.

  • +4 points per correct answer, −1 point per wrong answer
  • Calculates score percentage, accuracy, and avg time per question
  • Edit and delete any mock entry

🔄 4. Spaced Revision Scheduler

Schedule topic revisions using a proven spaced-repetition gap system. Maximum 6 revisions per topic.

  • Revision gaps: +3 → +7 → +15 → +30 → +60 → +120 days
  • Red highlight for overdue topics
  • +10 points per revision completed
  • Progress bar and due date shown per topic
  • Completed topics moved to a separate archived section

🔥 5. Streak & Consistency System

A streak counts consecutive days where both conditions are met:

  • 10+ questions in each subject (Physics, Chemistry, Maths)
  • At least 5 tasks completed that day
  • Green Days counted separately for practice-only consistency

📊 6. Points-Based Productivity Score

Your total score is a sum of all activity points:

  • Practice: +5 × questions solved
  • Mocks: +4 per correct, −1 per wrong
  • Revisions: +10 per completed revision
  • Tasks: +100 per completed task, −50 per undone task

📈 7. Stats Dashboard — Charts & Analytics

A rich analytics dashboard powered by Chart.js with 9 unique visualizations:

  • 📉 Task Completion Trend — daily/monthly line graph
  • 📉 Practice Trend — questions over time (7d / 30d / 1yr / all)
  • 🍩 Task Breakdown Pie — by status (done/pending) or by #tags
  • 📊 Task Stats Cards — total completed, completion %, avg/day, undone count
  • 📉 Mock Score Trend — score percentage across last 6 mocks
  • 📉 Time Taken Trend — minutes per mock test
  • 🗓️ Monthly Task Heatmap — calendar view of daily task completions
  • 🕸️ Productivity Radar — 5-metric normalized score (Tasks, Questions, Mocks, Revisions, Consistency)
  • 📈 Smoothed Progress Trend — 7-day rolling average of daily performance score

🌗 8. Dark / Light Mode

All charts, cards, and UI elements fully adapt to your chosen theme. Toggle anytime from the Settings tab.

🔐 9. Authentication & Cloud Sync

Sign up / log in with email & password. All data syncs to Supabase in real time. Works offline too — local storage keeps everything intact between sessions.

💾 10. Backup & Restore

  • Export all data as a .json backup file
  • Import a backup to restore everything instantly
  • Full data reset option available in Settings

🌟 Stay Consistent, Stay Ahead

Consistency is the only shortcut. Log every question, complete every task, revise on schedule — and watch your score climb. You've got this. 💪
